# US County Health Dashboard
### Unit 3 Group Project — CDC PLACES Data

## Overview

This project explores county-level chronic disease patterns across the United States using the
[CDC PLACES dataset](https://data.cdc.gov/500-Cities-Places/PLACES-Local-Data-for-Better-Health-County-Data-20/swc5-untb).

**Live dashboard:** *https://quynhanhtran.shinyapps.io/county-health-dashboard/*

## How to Run Locally

### Prerequisites

Install R (≥ 4.2) and RStudio, then install required packages:

```r
install.packages(c(
  "shiny",
  "shinydashboard",
  "tidyverse",
  "janitor",
  "skimr",
  "corrplot",
  "scales",
  "plotly",
  "DT"
))
```

### Step 1: Download the data

Open `eda-cp2.qmd` in RStudio and run the first code chunk. This will download the CDC PLACES
CSV and save it to `data/places_county.csv`. You only need to do this once.

### Step 2: Run the EDA notebook (optional)

Render `eda-cp2.qmd` to HTML with:

```r
quarto::quarto_render("eda-cp2.qmd")
```

### Step 3: Run the Shiny app

```r
shiny::runApp("app/")
```

### Step 4: Render the report

```r
quarto::quarto_render("report/report.qmd")
```

## Data Source

- **Dataset:** PLACES: Local Data for Better Health, County Data 2024 Release
- **Publisher:** Centers for Disease Control and Prevention
- **URL:** https://data.cdc.gov/500-Cities-Places/PLACES-Local-Data-for-Better-Health-County-Data-20/swc5-untb
- **License:** US Government open data (public domain)
- **Ethical note:** This dataset contains only aggregate county-level estimates. No
  individual-level records are present. No scraping was performed.
